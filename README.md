# Leaky Nodes


Der Mod »Leaky« (leaky) enthält Blöcke, die angelehnt an einige Standard-Nodes sind. Als Besonderheit kann man aber durch die Blöcke hindurchgehen (direkt oder per Mese-Schaltung).

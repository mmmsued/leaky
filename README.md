# Leaky Nodes

**Description:** The mod »Leaky« (leaky) contains blocks that are based on some textures from the mod »default« (»minetest_game«). However, you can simply walk through some of these blocks and make others temporarily invisible using Mesecons circuits.  

**Usage:** Search the inventory for »direct« for blocks that you can walk through directly (inventory image with a footprint) or for »ghost« for blocks that can be made invisible with Mesecons (inventory image with a yellow dot). In addition to stone, glass, leaf and grass blocks, there are also water and lava blocks with animated textures for both variants.  

**Known Issues:** There are currently no recipes for the blocks. Therefore, they can only be accessed with »giveme« or in creative mode.
